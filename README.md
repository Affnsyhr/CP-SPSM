# Backend Penerimaan Siswa Madrasah

Proyek ini adalah backend untuk sistem penerimaan siswa madrasah. Dibangun menggunakan Node.js, Express, dan PostgreSQL.

## Cara Menjalankan
1. Pastikan PostgreSQL sudah terinstal dan konfigurasi database sesuai dengan file `.env`.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Jalankan server:
   ```bash
   npm start
   ```

## Fitur
- Registrasi dan login pengguna
- Pendaftaran siswa
- Verifikasi siswa oleh admin
- Manajemen data siswa

## Langkah-Langkah Menggunakan GitHub Collaborator
1. Pastikan Anda memiliki akses ke repository proyek ini di GitHub.
2. Clone repository ke komputer lokal Anda:
   ```bash
   git clone "https://github.com/Affnsyhr/CP-SPSM.git"
   ```
3. Buat branch baru untuk fitur atau perubahan yang ingin Anda tambahkan:
   ```bash
   git checkout -b <nama-branch>
   ```
4. Lakukan perubahan pada kode sesuai kebutuhan.
`
git add .
`
6. Commit perubahan Anda:
   ```bash
   
   git commit -m "Deskripsi perubahan"
   ```
7. Push branch Anda ke repository:
   ```bash
   git push origin <nama-branch>
   ```
8. Buat Pull Request (PR) di GitHub dan tunggu hingga PR Anda direview dan di-merge oleh kolaborator lain.

9. Jika terjadi non-fast forward saat ingin push, coba gunakan 
```bash
git pull origin main --rebash
```
9. kemudian push  kembali !! happy coding 😊
```
git push origin main
```
